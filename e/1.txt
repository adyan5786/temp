1. Click 'New' button to create a New VM
2. Name = Ubuntu Server
3. Select existing hard disk and select .vdi file
4. Finish and machine ready
5. Click Settings, go to networking section, select NAT
6. Port Forwarding, add ports:
	- Host Port 2222 to Guest Port 22
	- Host Port 8080 to Guest Port 80
7. Start VM and Accept popups if any
8. Username: osboxes | Password: osboxes.org
9. Run following commands:
sudo apt update
sudo apt install nginx git -y
sudo service nginx status
cd /var/www/html
sudo rm -rf *
sudo git clone https://github.com/username/some-repo.git .
10. Open Chrome and go to http://localhost:8080
